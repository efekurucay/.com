  * **Cipher:** Cipher is an open-source, memory-powered AI agent framework specifically designed for coding agents. It offers a memory layer that helps AI agents scale with your codebase. It integrates seamlessly with popular IDEs and clients like Cursor, Windsurf, and VS Code through MCP (Model Context Protocol) integration. Cipher's dual memory layer captures both System 1 (programming concepts and business logic) and System 2 (the reasoning steps of the model) to provide a more consistent and robust coding experience. It also allows for the sharing of coding memories across development teams in real-time.
